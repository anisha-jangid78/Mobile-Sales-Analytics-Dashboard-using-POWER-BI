📊 Mobile Sales Dashboard | Power BI Project

📌 Project Overview

This project is a Power BI Mobile Sales Dashboard designed to analyze and visualize sales performance using interactive reports and data-driven insights.

The dashboard helps in understanding sales trends, customer behavior, and business performance.

🎯 Objectives

Analyze total sales, quantity, and transactions

Track monthly and city-wise performance

Understand customer ratings and payment methods

Compare brand-wise and model-wise sales

Create an interactive and user-friendly dashboard

📈 Key Features

✔ Total Sales, Quantity & Transactions Summary 

✔ Monthly Trend Analysis 

✔ Top Cities by Sales 

✔ Customer Rating Insights

✔ Payment Method Analysis 

✔ Brand & Model Performance 

✔ Interactive Filters & Slicers

🛠️ Tools & Technologies Used

Power BI

Power Query (Data Cleaning & Transformation)

DAX (Calculations)

Microsoft Excel (Data Source)
